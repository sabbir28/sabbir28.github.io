FirebaseAuthentication Java Class Documentation

Introduction

The FirebaseAuthentication class is a Java implementation for interacting with Firebase Authentication services. It provides methods to perform user registration, login, email verification, password reset, user deletion, and email change.

Dependencies

To use the FirebaseAuthentication Java class, you need the following dependencies:

  • Java Development Kit (JDK)
  • The org.json library for handling JSON data.

Usage

To use the FirebaseAuthentication class, follow these steps:

1. Create an instance of FirebaseAuthentication

FirebaseAuthentication firebaseAuth = new FirebaseAuthentication("YOUR_API_KEY");

2. User Registration

JSONObject registrationResult = firebaseAuth.createUser("user@example.com", "password");

Example: 

if (registrationResult.getBoolean("success")) {
    String idToken = registrationResult.getString("idToken");
    String email = registrationResult.getString("email");
    String refreshToken = registrationResult.getString("refreshToken");
    int expiresIn = registrationResult.getInt("expiresIn");
    String localId = registrationResult.getString("localId");
    System.out.println("Registration Successful!");
    System.out.println("ID Token: " + idToken);
    System.out.println("Email: " + email);
    System.out.println("Refresh Token: " + refreshToken);
    System.out.println("Expires In: " + expiresIn);
    System.out.println("Local ID: " + localId);
} else {
    String errorCode = registrationResult.getString("error_code");
    String errorMessage = registrationResult.getString("error_message");
    System.out.println("Registration Error: " + errorCode + " - " + errorMessage);
}

3. User Login

JSONObject loginResult = firebaseAuth.loginUser("user@example.com", "password");

Example: 

if (loginResult.getBoolean("success")) {
    String idToken = loginResult.getString("idToken");
    String email = loginResult.getString("email");

    String errorCode = loginResult.getString("error_code");
    String errorMessage = loginResult.getString("error_message");
    System.out.println("Login Error: " + errorCode + " - " + errorMessage);
}

4. Email Verification

JSONObject verificationResult = firebaseAuth.sendEmailVerification("ID_TOKEN");

Important Notes

- Replace "YOUR_API_KEY" with your Firebase project's API key. - Ensure your Java project includes the org.json library.

Conclusion

This Java class allows you to integrate Firebase Authentication into your Java applications. Refer to the method descriptions and error handling guidelines for effective usage.
